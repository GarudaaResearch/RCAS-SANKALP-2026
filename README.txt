SANKALP 2026 — RCAS Innovation & Startup Cell
==============================================
Website Package

FILES:
  index.html          — Main website page
  css/style.css       — All styles
  js/main.js          — Animations & interactions
  images/rcas-logo.png — College logo

HOW TO USE:
  1. Open index.html in any modern browser to view locally
  2. To host online, upload all files/folders to any web host (e.g. Netlify, GitHub Pages)
  3. Make sure folder structure is preserved (css/, js/, images/)

HOSTED ON NETLIFY (FREE):
  1. Go to https://netlify.com → Sign up
  2. Drag and drop the 'sankalp2026' folder
  3. Your site goes live instantly!

HOSTED ON GITHUB PAGES (FREE):
  1. Create a GitHub repo
  2. Upload all files
  3. Go to Settings → Pages → Deploy from main branch
